http://jpa.kapsi.fi/nanopb/

2 stages to produce source files:
	>  protoc -omyprotofile.pb myprotofile.proto
	>  python nanopb/generator/nanopb_generator.py myprotofile.pb

Nanopb runtime library:
	> pb.h
	> pb_common.h and pb_common.c (always needed)
	> pb_decode.h and pb_decode.c (needed for decoding messages)
	> pb_encode.h and pb_encode.c (needed for encoding messages)

Generated Files:
	> myprotofile.proto (just an example)
	> myprotofile.pb.c (autogenerated, contains initializers for const arrays)
	> myprotofile.pb.h (autogenerated, contains type declarations)
